Pollen Publisher Tools will add recommendation, realtime, analytics widgets to Wordpress post and pages. To create account, go to https://getpollen.co/

== Description ==

You can improve the default WordPress to include:

- add articles recommendation widget.
- articles realtime stats widget.
- articles analytics widget.

= Recommendation Widget =
Add post recommendation to each post's footer (if enabled in WordPress configuration).

= ADs =
Very simple implementation of advertisement to each recommendation widget. Ad can be configured on https://getpollen.co/.

= Support =

Please, if you need support or help, email support@getpollen.co

Included translations:

- English (default)

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Pollen Publisher Tools is now updated, you can edit the settings under the new "Pollen" menu.

No more settings are needed.
